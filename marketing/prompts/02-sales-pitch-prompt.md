# Prompt: Create a Sales Pitch for RubriCheck

Use this prompt with Claude to generate a sales pitch script:

---

You are a sales consultant creating a **sales pitch script** for **RubriCheck**, an AI-powered rubric grading platform.

**Product summary:**
- RubriCheck automates and standardizes grading with rubrics
- Teachers upload papers (PDF), define rubrics with criteria, and grade with AI-assisted workflows
- Integrates with Moodle for quizzes and LMS
- Provides dashboards for class performance, grade distribution, and subject-level insights
- Reduces grading time while improving consistency and feedback quality

**Target buyers:** School administrators, department heads, curriculum directors, EdTech decision-makers.

**Pitch requirements:**
- 2–3 minute spoken script (roughly 300–450 words)
- Structure: Hook → Problem → Solution → Key benefits → Social proof placeholder → Call to action
- Conversational tone, suitable for a live or recorded sales call
- Include 2–3 specific pain points (e.g., grading burnout, inconsistent feedback, manual data entry)
- Include 2–3 clear differentiators (e.g., rubric flexibility, Moodle integration, AI-assisted workflows)
- End with a soft CTA (demo request, pilot program, or contact)

**Output format:** Plain text script with optional [PAUSE] or [EMPHASIZE] cues for delivery.
