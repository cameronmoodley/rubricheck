# Prompt: Create a Pitch Deck for RubriCheck

Use this prompt with Claude to generate a pitch deck:

---

You are a marketing strategist creating a pitch deck for **RubriCheck**, an AI-powered rubric grading platform for educators.

**Product context:**
- RubriCheck helps teachers grade student papers and exams using customizable rubrics
- AI-assisted grading via n8n workflows (paper uploads, exam projects, Moodle quiz integration)
- Teachers create rubrics with criteria, upload papers (PDF), grade with structured feedback
- Integrates with Moodle for quiz grading and LMS workflows
- Dashboard with class performance, grade distribution, subject averages
- Multi-user: admins, teachers, students with role-based access

**Target audience:** K-12 schools, universities, training providers, anyone assessing written work at scale.

**Deck requirements:**
- 10–12 slides
- Include: Problem, Solution, How it works, Key features, Integrations (Moodle, n8n), Target market, Business model placeholder, Team/vision, Ask/next steps
- Professional, education-sector appropriate tone
- Use bullet points and slide titles; keep each slide concise for presentation
- Emphasize time savings, consistency, and data-driven insights for educators

**Output format:** Markdown with clear slide breaks (---) and slide numbers. Each slide should have a title and 3–5 bullet points max.
