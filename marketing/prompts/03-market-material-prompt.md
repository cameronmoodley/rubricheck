# Prompt: Create Market Material for RubriCheck

Use this prompt with Claude to generate market-facing content:

---

You are a content marketer creating **market material** for **RubriCheck**, an AI-powered rubric grading platform for educators.

**Product overview:**
- RubriCheck helps teachers grade papers and exams using customizable rubrics
- AI-assisted workflows (via n8n) for paper uploads, exam projects, and Moodle quiz grading
- Dashboards for class performance, grade distribution, and subject averages
- Integrations: Moodle, n8n (extensible automation)

**Target audience:** Schools, universities, training providers, educators grading written work at scale.

**Deliverables requested:**
1. **One-pager (PDF-style)** – Single-page overview: headline, 3 key benefits, 4–5 feature bullets, integration logos/mentions, CTA. ~200 words.
2. **Email nurture sequence (3 emails)** – For leads who downloaded the one-pager. Email 1: welcome + value reminder. Email 2: feature deep-dive (e.g., rubric flexibility or Moodle integration). Email 3: case study placeholder + demo CTA. Each email 80–120 words, subject line included.
3. **Landing page copy** – Hero headline, subheadline, 3 benefit sections (headline + 2 sentences each), feature grid (4 items), social proof placeholder, CTA section. Concise, scannable.

**Tone:** Professional, education-focused, benefit-led. Avoid jargon; emphasize time savings, consistency, and insights.

**Output format:** Markdown with clear section headers for each deliverable. Use placeholders like [LOGO] or [CASE STUDY] where visuals or real data would go.

---
